## 🎯 **游戏流程**

### **1. 开始游戏**
- 打开浏览器访问游戏
- 自动进入Start Town

### **2. 获得初始怪物**
- 走到地图中间
- 与Professor Oak对话
- 获得Flambear（火系初始怪物）

### **3. 探索和战斗**
- 在草丛中走动
- 遭遇野生怪物
- 进入战斗

### **4. 捕捉怪物**
- 战斗中削弱怪物HP
- 点击"Catch"按钮
- 成功捕捉后加入队伍

### **5. 挑战训练师**
- 在Route 1遇到训练师
- 按空格键发起挑战
- 击败获得胜利

## ⚔️ **战斗系统**

### **属性克制**
- 🔥 火 克 🌿 草
- 💧 水 克 🔥 火
- 🌿 草 克 💧 水
- ⚡ 电 克 💧 水 和 🦅 飞行

### **伤害公式**
```
伤害 = 技能威力 × (攻击/防御) × 属性克制 × 随机系数
```

### **捕捉机制**
```
捕捉率 = (1 - HP比例) × 0.5 + 0.2
```
- HP越低，捕捉率越高
- 基础捕捉率20%
