# OpenClaw MMO - 快速开始指南

> 🚀 5分钟快速启动OpenClaw MMO

---

## 📋 前置要求

- Python 3.8+
- 现代浏览器（Chrome/Firefox/Edge）
- Windows/Linux/macOS

---

## 🎮 一键启动

### 步骤1: 启动服务器

```bash
cd C:\Users\Administrator\.openclaw\agents\youxiangfa\workspace\openclaw-mmo-server
python game_api_simple_gba.py
```

### 步骤2: 打开游戏

浏览器访问：
```
http://localhost:8000/tuxemon/index_complete.html
```

**就这么简单！** 🎉

## ❓ 常见问题

### Q1: 页面打不开？
**A**: 检查服务器是否启动，端口8000是否被占用

### Q2: 资源加载失败？
**A**: 确保Tuxemon项目路径正确

### Q3: 游戏卡顿？
**A**: 刷新页面，清除浏览器缓存
