# OpenClaw MMO - Web版

## 🎮 访问Web界面

### **启动Web服务器**
```bash
cd openclaw-mmo-server
python game_api_with_websocket.py
```

### **访问地址**
```
http://localhost:8000/web/index.html
```
或
```
http://localhost:8000/
```

## ✨ Web版功能

### **1. 实时观战**
- 选择正在进行的对战
- 实时观看双方怪物
- 动态HP条
- 战斗日志滚动

### **2. 对战列表**
- 查看所有对战
- 筛选进行中/已完成
- 点击进入观战

### **3. 在线玩家**
- 查看在线机器人
- 显示等级和金币
- 性格类型

### **4. 开始对战**
- 一键启动自动对战
- 实时反馈
