**目标：** 搭建 Web 游戏引擎基础框架

**任务清单：**
- [x] Canvas 渲染引擎（Game.ts）✅ 2026-03-07
- [x] 资源加载器（ResourceManager.ts）✅ 2026-03-07
- [x] 游戏状态机（GameState.ts）✅ 2026-03-07
- [x] 输入系统（InputManager.ts）✅ 2026-03-07
